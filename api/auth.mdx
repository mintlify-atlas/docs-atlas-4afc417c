The base `Auth` class that all authentication methods inherit from. This class provides core authentication functionality and token management.

## Class Signature

```python
class Auth:
    def __init__(self, **kwargs: Any) -> None
```

## Parameters

<ParamField path="ios_token" type="str" optional>
  The main Fortnite token to use with authentication. You should generally not need to set this manually.
  
  Default: `M2Y2OWU1NmM3NjQ5NDkyYzhjYzI5ZjFhZjA4YThhMTI6YjUxZWU5Y2IxMjIzNGY1MGE2OWVmYTY3ZWY1MzgxMmU=`
</ParamField>

<ParamField path="device_id" type="str" optional>
  A 32 char hex string representing your device. Auto-generated if not provided.
</ParamField>

<ParamField path="token_type" type="str" optional>
  The access token type. It's recommended you only change this if you know what you're doing as certain functions/API calls may start causing errors.
  
  Default: `eg1`
</ParamField>

## Properties

<ResponseField name="authorization" type="str">
  The Authorization header for use with Fortnite endpoints. Alias for `ios_authorization`.
</ResponseField>

<ResponseField name="ios_authorization" type="str">
  Returns `bearer {ios_access_token}` for iOS authentication.
</ResponseField>

<ResponseField name="chat_authorization" type="str">
  Returns `bearer {chat_access_token}` for chat authentication.
</ResponseField>

<ResponseField name="eas_authorization" type="str">
  Returns `bearer {eas_access_token}` for EAS authentication.
</ResponseField>

<ResponseField name="eos_authorization" type="str">
  Returns `bearer {eos_access_token}` for EOS authentication.
</ResponseField>

<ResponseField name="identifier" type="str">
  Returns the authentication identifier. Must be implemented by subclasses.
</ResponseField>
